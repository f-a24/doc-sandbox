What is React

FacebookがOSSとして公開しているUIを構築するためのJavaScriptライブラリ

A JavaScript library for building user interfaces

https://reactjs.org/

Concept

  • Declarative
    宣言型
  • Component-Based
    コンポーネントベース
  • Learn Once, Write Anywhere
    一度学び、どこにでも書く

Declarative

高階コンポーネントなど関数型の思考

Component-Based

UIをコンポーネント(部品)として構築

Learn Once, Write Anywhere

Webだけではなくモバイルアプリ、VR等も可能

  • React Native
  • React360

Advantage

  • 仮想DOMという作りで画面を差分更新することによって描画が速い
    （時と場合によりけり）
  • コンポーネント単位で作っていくため管理がしやすい

Therefore…

  • HTMLを操作するためのものではない
  • サーバー側はデータを返すだけのような仕組みにする必要がある
  • jQueryとの併用も可能だが好ましくはない

How to use

App.jsx

export default class App extend React.Component {
  constructor(props) {
    super(props);
  }
  render() {
    return (
      <div>
        <h1>Hello, world!</h1>
      </div>
    );
  }
}

Component

Statefull Component

class App extend React.Component {
  constructor(props) {
    super(props);
    this.state = { data: '' };
    this.inputHandler = this.inputHandler.bind(this);
  }
  inputHandler (e) {
    this.setState({ data: e.target.value });
  }
  render() {
    return (
      <div className="msg">
        <p style={{color: 'skyblue'}}>{this.state.data}</p>
        <Input onChange={this.inputHandler} />
      </div>
    );
  }
}
